This project includes
1. Authentication using JWT
2. Creating rest API's using node and express
3. Accessing restAPI's
4. version management of API's
5. Acces Mongo DB
6. Logging 
7. Clustering to take advantage of multi core 

Other concepts to include
1. Use Reddis/memcached cache for token management
2. Use yaml for configurations as its superset of JSON
3. Using other authentication mechanism like Okta.(https://developer.okta.com/quickstart/#/angular/nodejs/express)
https://developer.okta.com/blog/2017/10/19/use-openid-connect-to-build-a-simple-node-website
4. Authentication can be managed by token or session management
5. At the infrastructure level, we can use deployment manager tools like PM2, Docker Compose and Kubernetes to specify the environment.
